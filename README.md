# berkeley-homes
Near Miss &amp; Positive Intervention

## Setup:
  1. Clone this repository
  2. Execute `npm i`

## Build:
none

## Test:
  1. Execute `npm test`

## Run
  1. create `.env` file in project routes
  2. add aws tokens
    AWS_ACCESS_KEY_ID=****
    AWS_SECRET_ACCESS_KEY=****
  3. add your s3 bucket name
    BUCKET_NAME=****
    **nb** This defaults to `berkeley-homes-near-miss`
  4. Execute `npm run`
